Long Questions

These questions require full proofs.

\begin{framed} There is actually a problem with the way this question was phrased on the exam (my apologies!). Going from $y$ to a system is essentially fine. It's going back the other way that is the problem. The question should have read as follows: \end{framed}
  1. Show that solutions of the second order, linear, constant coefficient homogeneous equation \begin{equation} \label{eq:2olcch} ay'' + by' + cy = 0 \end{equation} with \(a\ne 0\) correspond to solutions of the first order, linear, homogeneous system of equations \begin{equation} \label{eq:1olhs} \begin{pmatrix} x_1' \\ x_2' \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ \alpha_1 & \alpha_2 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix}. \end{equation}
    1. Show that given a solution \(y\) to \(ay'' + by' + cy = 0\), setting \(x_1 = y\), \(x_2 = y'\) one obtains a solution of the system

      \begin{equation} \label{eq:1olhs} \begin{pmatrix} x_1' \\ x_2' \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ \alpha_1 & \alpha_2 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} \end{equation} with \(\alpha_1 = -c/a\), \(\alpha_2 = -b/a\).

      Conversely, given \((x_1,x_2)\) a solution of the above system, show that setting \(y=x_1\), one obtains a solution of \[ y'' - \alpha_2 y' - \alpha_1 y = 0. \]

      \begin{framed} To get from a solution $y$ to a system $(x_1, x_2)$, let $y$ be a solution of $ay'' + by' + cy = 0$ and as suggested, let \begin{align*} x_1 &= y \\ x_2 &= y'. \end{align*} Then \[ x_1' = y' = x_2 \] and since $ay'' + by' + cy = 0$, we have \[ x_2' = (y')' = y'' = -\tfrac{b}{a} y' - \tfrac{c}{a} y = -\tfrac{b}{a} x_2 - \tfrac{c}{a} x_1. \] These two equations together show that $(x_1,x_2)$ is a solution of the system \begin{equation} \begin{pmatrix} x_1' \\ x_2' \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ -c/a & -b/a \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} \end{equation} which is of the required form by setting $\alpha_1 = -c/a$ and $\alpha_2 = -b/a$. To get from a system $(x_1, x_2)$ to $y$, set $y=x_1$. Then $y' = x_1' = x_2$ since $x_1'=x_2$ from the system and thus also $y'' = (y')' = x_2'$. Now observe that from the system \[ x_2' = \alpha_1 x_1 + \alpha_2 x_2 = \alpha_1 y + \alpha_2 y' \] or \[ y'' - \alpha_2 y' - \alpha_1 y = 0 \] as required. \end{framed}

    2. Show that the constructions above are inverse to each other.

      \begin{framed} Start with a solution $y$ and produce the corresponding system $(x_1, x_2) = (y, y')$ with $\alpha_1 = -c/a$, $\alpha_2 = -b/a$. Now take the system and produce a new function $\bar{y} = x_1$ which is equal to $y$ by construction of $x_1$. That is, converting $y$ to a system and then back to a function we just get $y$ again. For the other way round, take a system $(x_1,x_2)$ and produce a function $y = x_1$. Then convert to a new system $(\bar{x}_1, \bar{x_2}) = (y, y')$. But now we have \[ (\bar{x}_1, \bar{x_2}) = (y, y') = (x_1, x_1') = (x_1, x_2) \] since we already know from the original system that $x_1'=x_2$. Thus converting a system to a function and then to a system recovers the original system. \end{framed}

  2. (10 points) In class we studied a spring mass system without a forcing term, i.e. a homgeneous equation. Suppose now, we drive the spring back and forth (for instance with a motor) with amplitude \(R\) and frequency \(\omega\) (in some appropriate units). This can be modelled as follows: \[ m u'' + \gamma u' + k u = R \sin (\omega t) \] with \(m,k,\gamma > 0\). Find the general solution of this equation, in the undamped situation (\(\gamma = 0\)) as follows:
    1. Show that the general solution of the homogeneous equation is \[ y(t) = C_1 \cos(\omega_0 t) + C_2 \sin(\omega_0 t) \] where \[ \omega_0^2 = \frac{k}{m}. \] \begin{framed} The characteristic equation is $m r^2 + k = 0$ so that \[ r = \sqrt{-\frac{k}{m}} = i \sqrt{\frac{k}{m}} \] since $m,k\geq 0$. Thus the roots are pure imaginary and the solution is \[ y(t) = C_1 \cos \left(\sqrt{\frac{k}{m}} t\right) + C_2 \sin \left(\sqrt{\frac{k}{m}} t\right). \] \end{framed}
    2. Using the method of undetermined coefficients (or variation of parameters if you prefer), show that a particular solution \(Y\), to the non-homogeneous equation is given by \[ Y(t) = \frac{R}{k - m\omega^2} \sin \omega t. \] \begin{framed} Let $Y(t) = A \sin \omega t + B \cos \omega t$. Then for $Y$ to be a particular solution we must have \[ \begin{split} R\sin \omega t &= m Y'' + k Y = m\left(-A \omega^2 \sin \omega t - B \omega^2 \cos \omega t\right) + k \left(A \sin \omega t + B \cos \omega t\right) \\ &= A (k - m \omega^2) \sin \omega t + B (k - m \omega^2) \cos \omega t. \end{split} \] Equating coefficients on the left hand side with right hand side gives \begin{align*} A(k - m \omega^2) &= R \\ B(k - m \omega^2) &= 0 \end{align*} so that $B=0$ and $A = r/(k-m\omega^2)$ and so \[ Y(t) = \frac{R}{k - m\omega^2} \sin \omega t. \] \end{framed}
    3. Write down the general solution.

      \begin{framed} The general solution is the sum of the general solution of the homogeneous equation plus a paritcular solution, hence is \[ y(t) = C_1 \cos(\omega_0 t) + C_2 \sin(\omega_0 t) + \frac{R}{k - m\omega^2} \sin \omega t \] for arbitrary constants $C_1,C_2$. \end{framed}

    4. In the presence of damping (\(\gamma > 0\)) the situation is more complex. However, we can obtain some qualitative information easily. Suppose, the system is under damped (\(\gamma^2 - 4mk < 0\)) so that the homogeneous equation has solution \[ e^{-\gamma t/2m} \left[C_1 \cos(\mu t) + C_2 \sin(\mu t)\right]. \] Show that as \(t\to \infty\) the general solution approachs a steady state solution of the form \[ U(t) = A \sin \omega t + B \cos \omega t. \] Thus the damping dissipates the solution of the homogeneous equation (called a transient state) and the solution approaches (exponentially fast) an oscillation with frequency given by the driving frequency \(\omega\).

      \begin{framed} The general solution of the non-homogeneous equation is the sum of the general solution to the homogeneous equation plus a particular solution. Let $U$ be a particular solution so that \[ y(t) = e^{-\gamma t/2m} \left[C_1 \cos(\mu t) + C_2 \sin(\mu t)\right] + U(t). \] Since $\gamma>0$ the homogeneous equation approaches $0$ as $t\to \infty$ and so \[ \lim_{t\to\infty} \left(y(t) - U(t)\right) = 0. \] Variation of parameters now ensures that we can find a particular solution of the form \[ U(t) = A \sin \omega t + B \cos \omega t. \] as required. \end{framed}

Short Answer

These questions only require brief justification. No points will be given for answers without justification.

  1. (1 point) Let \(y\) be a solution of \(ay'' + by' + cy = 0\) with \(a\ne 0\). The phase plot of \(y\) is the curve \((y(t), y'(t))\) in the plane. If \(y_1\) and \(y_2\) are different solutions, can their phase plots intersect?

    \begin{framed} This question may have been a little too subtle. The answer is yes. Two different solutions can have phase plots that intersect, though they must do so at different times. That is, we can't have $(y_1(t_1), y_1'(t_1)) = (y_2(t_1), y_2'(t_1))$ however, it could be that $(y_1(t_1), y_1'(t_1)) = (y_2(t_2), y_2'(t_2))$ for $t_1\ne t_2$. For instance $y_1(t) = \cos t$ and $y_2(t) = \sin t$ are different solutions of $y'' + y = 0$, but both phase plots are the circle of radius $1$ centered on the origin in the $(y,y')$-plane. The phase plots both loop around clockwise as $t$ increases. At $t=0$, $(y_1(0), y_1'(0)) = (1, 0)$ and $(y_2(0), y_2'(0)) = (0, 1)$ so that $y_2$ is always "behind" $y_1$ by 90 degrees. This behaviour is true in general. If $y_1, y_2$ are different solutions whose phase plots intersect then the path traced by both phase plots is the same (which is what I had in mind for this question), even though the phase plots are never at the same point at the same time. \end{framed}

  2. (1 point) In the same situation as the previous question, can the curves \((t, y_1(t))\), \((t, y_2(t))\) (i.e. the graphs of \(y_1\) and \(y_2\)) cross?

    \begin{framed} Yes, provided that their derivatives are different where they cross. \end{framed}

  3. (1 point) Which of the following is the first order system corresponding to the equation \(u'' + 1/2 u' + 2u = 0\)?
    1. \[ \begin{split} x_1 &= u \\ x_2 &= -1/2 x_2 - 2x_1 \end{split} \]
    2. \[ \begin{split} x_1' &= x_2 \\ x_2' &= -1/2 x_1 - 2x_1 \end{split} \]
    3. \[ \begin{split} x_1' &= x_2 \\ x_2' &= -1/2 x_2 - 2x_1 \end{split} \] \begin{framed} The answer is (c). The system is given by $x_1=u$, $x_2=u'$ so that $x_1'=u' = x_2$ and $x_2' = u'' = -1/2 u' - 2u = -1/2 x_2 - 2 x_1$. \end{framed}
  4. (1 point) Suppose the characteristic equation \(ar^2 + br + c = 0\) has a repeated root \(r \ne 0\). Is \[ y(t) = C_1 e^{rt} + C_2 e^{-rt} \] a solution of the homogeneous equation \(ay'' + by' + cy = 0\)?

    \begin{framed} No it is not. $-r$ is not a root of the characteristic equation since $r$ is repeated and $-r\ne r$ because $r\ne 0$. \end{framed}

  5. (1 point) If \(n\) solutions \(x_1, \dotsc, x_n\) to a linear first order system of \(n\) equations in \(n\) unknowns are linearly independent at \(t_0\), are they linearly independent at \(t_1\ne t_0\)?

    \begin{framed} Yes! If they are linearly independent at $t_0$, then W(t_0) \ne 0$. Since $W(t)$ is either zero for every $t$, or never zero we must have $W(t_1)\ne 0$ so that the solutions are linearly independent. \end{framed}
