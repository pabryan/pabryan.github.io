Lecture 4.2 Notes

Eigenvalues and eigenvectors

  • Try solutions of \(\vec{x}' = A \vec{x}\) of the form \(e^{rt} \vec{u}\).
  • This leads to the eigenvector equation \(A\vec{u} = r \vec{u}\)

The characteristic equation

  • We can solve for a non-zero \(\vec{u}\) if and only if \(0 = \det (A - rId)\)
  • This is called the characteristic equation. Expanding out the determinant, it is a quadratic equation for \(r\).

Real, distinct roots

  • When the roots \(r,s\) are real and distinct, we get eigenvectors \(\vec{u}, \vec{v}\).
  • Substitute the value of \(r\) into \((A - rID)\vec{u} = 0\) to find \(\vec{u}\). It always has an arbitrary constant. Using \(s\), we get \(\vec{v}\).
  • These solutions \(e^{rt} \vec{u}\) and \(e^{st}\vec{v}\) have non-zero Wronskian so the they form a fundamental set of solutions.

Complex roots

  • Do everything the same as for the real case, but now we use complex numbers \(r = \lambda + i \mu\).
  • The resulting \(\vec{u}\) will be complex.
  • We only need to use \(r\) and \(\vec{u}\). The other solution (corresponding to the other root \(s\)) is the complex conjugate of the one we found.
  • Multiply out \(e^{rt} \vec{u}\), using Euler's formula and separate the result into real and imaginary parts.
  • The Wronskian again is non-vanishing, so we have a fundamental set of solutions.
