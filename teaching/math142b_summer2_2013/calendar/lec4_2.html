Lecture 4.2 Overview

  • Sequences of functions Here we consider sequences of functions. That is for each natural number \(n\), we have a function \(f_n: D \to \mathbb{R}\). Note that \(D \subset \mathbb{R}\) is fixed here. We have the notion of pointwise convergence, which is just convergence at each point.
  • Examples We look at several examples to see how to calculate the pointwise limit of a sequence of functions.
  • Properties of the limit The limit function (if it exists) does not necessarily inherit properties such as continuity, differentiability and integrability from the sequence. Even when it does, the derivative of the limit is not necessarily the limit of the derivatives. The analogous statement also holds for integrals.

Lecture 4.2 Notes

Sequences of functions

  • \(f_n \to_{\text{pw}} f\) (read \(f_n\) converges pointwise to \(f\)) if for each fixed \(x \in D\), \[ \lim_{n\to\infty} f_n(x) = f(x) \] where \(\{f_n(x)\}\) is now just a sequence of real numbers.

Examples

  • The examples all came from the text, except one: \(f_n(x) = x^n/n\).

Properties of the limit

  • If each \(f_n\) is continuous, the limit need not be continuous.
  • If this statement were true, it would mean that \(\lim_{x\to x_0} f(x) = f(x_0)\). Since \[ \lim_{x\to x_0} f(x) = \lim_{x\to x_0} \lim_{n\to\infty} f_n(x). \] and \[ f(x_0) = \lim_{n\to\infty} f_n(x_0) = \lim_{n\to\infty} \lim_{x\to x_0} f_n(x), \] \(\lim_{x\to x_0} f(x) = f(x_0)\) amounts to saying that \[ \lim_{x\to x_0} \lim_{n\to\infty} f_n(x) = \lim_{n\to\infty} \lim_{x\to x_0} f_n(x). \] That is, it would mean that we can interchange the order limits, which is not true in general.
  • If each \(f_n\) is differentiable, the limit need not be differentiable.
  • If each \(f_n\) is integrable, the limit need not be integrable.
  • If each \(f_n\) is differentiable, and the limit \(f\) is differentiable, it is not true in general that \(f_n' \to_{\text{pw}} f'\). That is, we can't interchange the order in which we take limits and derivatives. From the definition of the derivative, this would amount to the statement that \[ \lim_{n\to\infty} \left[ \lim_{h\to 0} \frac{f_n(x) - f_n(x+h)}{h}\right] = \lim_{h\to 0} \left[\frac{\lim_{n\to\infty} f_n(x) - \lim_{n\to\infty} f_n(x+h)}{h}\right]. \] In other words, it would amount to saying that we can interchange the order in which we take the limits. This is not true in general.
  • Similarly, if each \(f_n\) is integrable, and the limit \(f\) is integrable, it is not true in general that \(\lim_{n\to\infty} \int f_n = \int f\). That is, we can't interchange the order in which we take limits and integrals.
