Lecture 2.1 Overview

This lecture deals mostly with the first and second fundamental theorems (6.5,6.6). Theorem 6.19 needs to be examined first. We need the result for the first fundamental theorem, but the proof is worth understanding and the result is more generally applicable than just for the first fundamental theorem.

Theorem 6.19 says that if a function \(f\) is continuous on \((a,b)\) and bounded on \([a,b]\) then it is integrable, with integral independent of the value of \(f\) at the endpoints.

The first fundamental theorem says that to evaluate an integral, one can find an anti-derivative.

The second fundamental theorem says that if \(f\) is continuous, then an anti-derivative always exists. It also tells us how to differentiate an integral.

Putting the two theorems together, we find that integrals and derivatives are inverse operations to each other.

Lecture 2.1 Notes

Theorem 6.19

This is quite an important theorem.

  • The only place continuity is used is to show that \(f\) is integrable on \([c,d]\) whenever \([c,d] \subset (a,b)\). Thus the proof actually shows that if a function \(f\) is integrable on \([c,d]\) whenever \([c,d] \subset (a,b)\), and bounded on \([a,b]\), then \(f\) is integrable on \([a,b]\) and the integral doesn't depend on the value at the endpoints.
  • From this theorem, we can easily prove that if \(c\in[a,b]\), then changing the value of \(f\) at \(c\) doesn't affect the integral. Hint: Use additivity over intervals.

Section 6.5

  • We needed Lemma 6.21. This says that if \(A\) is a real number such that \[ L(f,P) \leq A \leq U(f,P), \] for every \(P\), then \(A = \int f\). This follows since the integral is equal to both the best upper bound for all the \(L(f,P)\) and the best lower bound for all the \(U(f,P)\). The assumption here is that \(A\) is also an upper bound for the \(L(f,P)\) and a lower bound for the \(U(f,P)\). See the text to see that the only possibility is that \(A=\int f\).
  • To prove the first fundamental theorem, we choose any partition and apply the Mean Value Theorem on each partition interval. This says that there is a point \(c_i \in [x_{i-1}, x_i]\) such that \[ F(x_i) - F(x_{i-1}) = F'(c_i)(x_i - x_{i-1}). \]
  • To finish, we proved the inequality, \[ U(F', P) \leq F(b) - F(a) \leq L(F', P) \] for any partition, which we obtained by using the Mean Value Theorem and the bounds \[ m_i(F') \leq F'(c_i) \leq M_i(F') \] which lead to the required inequality, through the now very familiar telescoping sum. Then Lemma 6.21 completed the proof with \(A = F(b) - F(a)\) and \(f = F'\).
  • We also saw how integration by parts follows easily from the first fundamental theorem.

Section 6.6

  • For the first fundamental theorem, we used the Mean Value Theorem (for derivatives). The analogous result for integrals is the Theorem 6.26 (the Mean Value Theorem for integrals) which states that there is a point \(x_0 \in [a,b]\) such that \[ \frac{1}{b-a} \int_a^b f = f(x_0). \] The right hand side can be interperted as the mean value of \(f\) over \([a,b]\). We didn't prove this theorem. You should read the text for the proof which is reasonably straight forward.
  • Before proving the second fundamental theorem, we proved Proposition 6.27, which states that if \(f\) is integrable, then the function \(F(x) = \int_a^x f\) is continuous. The proof came from showing that \[ \vert F(u) - F(v) \vert \leq M \vert u - v\vert \] where \(|F| \leq M\). This latter result proves \(F\) is continuous. This is because if \(|u-v| \leq \tfrac{\epsilon}{M}\), then \(|F(u) - F(v)| \leq \epsilon\).
  • To prove the second fundamental theorem, we used the mean value theorem for integrals to show that \[ \lim_{x\to x_0} \left[ \frac{F(x) - F(x_0)}{x-x_0}\right] = f(x_0). \] The key observation was that the mean value theorem gave us a point \(c(x)\) strictly between \(x\) and \(x_0\) such that \(f(c(x))\) equals the term insise the brackets above. So \(c(x) \to x_0\) as \(x\to x_0\). Since \(f\) is continuous, \(f(c(x)) \to f(x_0)\) as \(x\to x_0\) completing the proof.
  • We also defined integrals in the "negative" direction \[ \int_d^c f = -\int_c^d f \] when \(c< d\), and \[ \int_c^c f = 0. \] With these definitions, additivity over intervals holds. In fact, if we want additivity over intervals to hold, we're forced into this defintion.
  • Let me also remark that we saw \(f\) integrable implies \(F\) continuous and \(f\) continuous implies \(F\) is differentiable. In fact if \(f\) is \(k\)-times differentiable, then \(F\) is \(k+1\)-times differntiable, which can be seen easily by induction. Thus integration adds one to number of times a function is differentiable, while differentations substracts one from this number. This should not be suprising from the fact that integration and differentiation are inverses to each other. Thus, even though derivatives are easier to compute than integrals, they make a function "worse" whereas integrals make a function "better".
