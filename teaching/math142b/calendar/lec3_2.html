Lecture 3.2 Notes

Approximating functions by Taylor Polynomials

  • If \(p_{\infty}(x) = f(x)\) then \(p_n(x)\) gives a good approximation of \(f\) for \(n\) sufficiently large.
  • \(p_{\infty}(x) = f(x)\) if and only if \(R_n(x) \to 0\) as \(n\to\infty\)
  • If \(p_{\infty}(x) = f(x)\), we say \(f\) is analytic.

Cauchy Integral Remainder Theorem

  • By using induction and integration by parts on the first fundamental theorem, we obtained a "higher order" first fundamental theorem.
  • This higher order first fundamental theorem gave the integral expression for \(R_n(x)\).

Comparison of Remainder Theorems

  • Lagrange Remainder Theorem
    • Comes from Cauchy Mean Value Theorem which is a higher order Mean Value Theorem that can be proven by induction.
    • We get an unknown function \(c(x)\)
    • We can still estimate \(R_n\).
    • This allowed us to prove that the exponential and tigonometric functions are anaylytic
  • Cauchy Integral Remainder Theorem
    • Comes from higher order first fundamental theorem, proven by induction.
    • We get an exact formula for \(R_n(x)\).
    • In general, the integrals can't be evaluated in any nice way.

Taylor Series for \(1/x\)

  • We calculated the derivatives of \(1/x\) to get \(p_n\) with \(x_0=1\).
  • The Lagrange Remainder Theorem does not help us to show \(R_n \to 0\)
  • We evaluated the integral to get an exact expression for \(R_n\).
  • This converges to \(0\) if and only if \(|x-1| < 1\) and so the radius of convergence is one.
  • We can calculate \(1/x\) to 3 of decimal places by finding \(n\) such that \(|R_n(x)| < 10^{-3}\) and the answer is then \(p_n(x)\).
  • We can do this for any desired number of decimal places.
