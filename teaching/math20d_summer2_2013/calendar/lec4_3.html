Lecture 4.3 Overview

  • Discontinuous forcing functions In many applications, the non-homogeneous term \(g\) may not be continuous. For instance, when turning on a power point (in an idealised way), there is an instentaneous jump from 0 volts to 110V. We may be able to solve the equation before the jump and after the jump, but the solution cannot be differentiable (for a first order equation) at the jump. Thus we need a rigorous method of studying such situations.
  • Laplace Transform The Laplace Transform is what is known as an integral transform. Given a function \(f=f(t)\), we define a new function \[ F(s) = \int_{\alpha}^{\beta} K(s, t) f(t) dt \] where the function \(K\) is a fixed function known as the kernel of the transformation. In the case of the Laplace Transform, \(K(s,t) = e^{-st}\).
  • Solving equations with the Laplace Transform The Laplace Transform takes a differential equation and transforms it into an algebraic equation, which may be more amenable to solving, particularly in the case of discontinuous forcing functions. The solution of this algebraic equation is then transformed back via the Inverse Laplace Transform to a solution of the differential equation. Much of the computations required can be undertaken by consulting a table of Laplace Transformations.

Lecture 4.3 Notes

Discontinuous forcing functions

  • If \(g\) is discontninous, then the "solution" may not be twice differentiable.
  • We considered the equation \(y'' + y = g\) where \(g\) is zero for \(0\leq t \leq \pi\) and one for \(\pi < t \leq 2\pi\). If \(y\) is a solution, then for \(0< t < \pi\) we have \(y = C_1 \cos t + C_2 \sin t\) and for \(0< t < \pi\) we have \(y = D_1 \cos t + D_2 \sin t\). For \(y\) to be continuous at \(t=\pi\) we needed \(D_1 = C_1 - 1\). For \(y'\) to be continuous at \(t=\pi\) we needed \(D_2 = C_2\). However, for \(y''\) to be continuous at \(t=\pi\) we needed \(D_1 =C_1\) contradicting that \(D_1 = C_1\).
  • Thus we can't find a two-times differentiable function \(y\) solving the equation.
  • However, we still all \(y\) a solution with aribtrary constants \(C_1,C_2\) since it's the closest we can get and from real-life considerations, we believe there should be a solution.

Laplace Transform

  • The Laplace Transform is an integral transform.
  • It is linear.
  • We calculated a few examples.
  • If \(f\) grows now faster than an exponential function \(Ke^{at}\) as \(t\to\infty\) the Laplace transform is defined for \(s>0\).
  • The Laplace Transform of zero is zero. Therefore (at least for our purposes), if two functions have the same Laplace transform, then they are in fact the same function.

Solving equations with the Laplace Transform

  • We calculated the Laplace transform of \(f'\) in terms of \(f\). Doing the same thing with \(f\) replaced by \(f'\) gave us the Laplace transform of \(f''\) in terms of \(f'\) and so it terms of \(f\).
  • Applying the Lalplace transform to a differential equation, and using the initial conditions, gives an expression for the Laplace transform of \(y\).
  • We need to find a function with the same Laplace transform.
  • In the example, we use a partial fraction decomposition to show that the a sum of exponentials works. In general, consult the table in the text.
  • Thus we found the solution and it agreed with the solution we calculated using the methods we already knew.
