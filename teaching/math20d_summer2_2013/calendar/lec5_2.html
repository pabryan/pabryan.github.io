Lecture 5.2 Overview

  • Step functions and their Laplace transform One of the main applications of the Laplace transform is in studying the effect of discontinuities in the driving force. We've already seen that for a second order equation, there cannot exist a solution with continuous second derivative. We need to develop methods of handling such situations, rather than taking an ad-hoc approach like the one described last week.

    We will consider "jump" discontinuities, which are modelled by step-functions. The first step is then to derive the Laplace transform of step functions.

  • Differential equations with discontinuous forcing functions With the Laplace transform able to handle jump discontinuities, we can apply it differential equations whose forcing term undergoes a jump discontinuity. Then we can solve such equations in precisely the same manner as we did last week using the Laplace transform.
